# DecentCoding (Preview)

A Roslyn analyzer to fix dirty words in C# code. 

Current support:

- Type name
- Property name
- Field name

## Usage

Add ```DecentCodingAnalyzer``` NuGet package to your C# project.

For .NET CLI
```
dotnet add package DecentCodingAnalyzer
```
